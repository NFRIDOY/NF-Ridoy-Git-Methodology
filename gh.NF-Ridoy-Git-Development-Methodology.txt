Init The Repo => …or create a new repository on the command line
---------------------------------------------------------
echo "# NF-Ridoy-Git-Development-Methodology" >> README.md
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/NFRIDOY/NF-Ridoy-Git-Development-Methodology.git
git push -u origin main

…or push an existing repository from the command line
-----------------------------------------------------------------------------------------

git branch -M main
git push -u origin main
